CROSS AXIS

Cross axis is the axis that is perpendicular to the main axis. In case your main axis is "row | row-reverse", cross axis will be "column | column-reverse" and vice versa.

START AND END LINES

Start and End lines are changeable lines that depends on the language's writing direction which you use in your device.

FLEX CONTAINER

display: flex | inline-flex;

INLINE FLEX

inline-flex property set the container an inline-block element, you can still set width,height or padding values like a block element has. It continues to behave like flex container.

FLEX WRAP

flex-wrap: no-wrap | wrap | wrap-reverse;

In order to use the "flex-wrap" property effectively, flex items (will be talked about later) should be too large to display them in one line.

Another usage of "flex-wrap" property is to protect the flex items' exact width. It can be used when we do not want to shrink them in the main axis.

FLEX GROW

flex-grow: number[0,∞]; (default 0)

Thanks to "flex-grow" property, we can grow the items proportionally from their "flex-basis" values.

flex-grow:2
flex-grow:1
flex-grow:1

As you see, the first element takes up bigger space than others.(2 times bigger.)

FLEX SHRINK

flex-shrink: number[0,∞]; (default 1)

"flex-shrink" property allows us to shrink items proportionally from their flex-basis values. It only works when the total size of the items overflows the main axis. Otherwise do not.

flex-shrink:4
flex-shrink:2
flex-shrink:1
